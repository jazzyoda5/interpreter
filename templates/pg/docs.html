About Tranescript

Tranescript is a multi purpose scripting language. The goal of the creator was to get recognition from ladies around the world, so for some reason he decided that building an interpreter was the way to go. Using Tranescript in your production environment is not recommended, unless you want to sabotage the company you work for, in which case it is probably your best choice.


Usage


  • Built In Types

    Tranescript supports three built in data types.
    Integer - int
    String - str
    Boolean - bool

  • print(args*)

    The print() function prints the given arguments to the standard output device.
