Usage


  • Built In Types

    TS supports three built in data types.
    Integer - int
    String - str
    Boolean - bool


  • Declaring Variables

    Variables should be declared like this
    [var name]: [var type] = [value];
    Example: var: str = 'I am a string';


  • Arithmetic Operations

    TS supports addition, subtraction, division and multiplication.

    Addition - +
    Subtraction - -
    Division - /
    Multiplication - *

    You can also wrap terms in parenthesis like so - (3 + (2 - 1)) * 4


  • print(args*)

    The print() function prints the given arguments to the standard output device.


  • Functions

    Functions must be defined using the keyword function. You can declare as many arguments as you want, but you must also declare the type.

    exmaple: function [function name]([arg name]: [arg type]).


  • Conditional Blocks

    If - else blocks can be defined like this:
    example: if ([condition]) { [block] } else { [block] }

    Given condition can be a Boolean value or a comparison.
    Supported comparison types are: >, <, /, *, <=, >=. Comparsion is only tested on int data types.
